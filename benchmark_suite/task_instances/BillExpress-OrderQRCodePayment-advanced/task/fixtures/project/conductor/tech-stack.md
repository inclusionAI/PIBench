# Tech Stack

## Language
- **TypeScript**: End-to-end type safety for rock-solid reliability across frontend and backend.

## Frontend
- **React 19**: Leverages the latest React paradigms for a highly interactive UI.
- **Vite 6**: Lightning-fast HMR and optimized production builds.
- **Tailwind CSS (v4)**: Rapid, atomic CSS styling.

## Backend
- **Express.js (Node.js)**: Lightweight web framework handling API routes.

## Database
- **SQLite3**: Incredibly fast, zero-configuration local database using `better-sqlite3`.
