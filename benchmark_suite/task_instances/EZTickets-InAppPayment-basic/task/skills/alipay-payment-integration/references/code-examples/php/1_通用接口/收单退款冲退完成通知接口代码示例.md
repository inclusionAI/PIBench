# 收单退款冲退完成通知接口 - PHP 示例

## 请求示例

```php
<?php
require_once '../aop/AopClient.php';

// 初始化验签客户端
$aop = new AopClient();
$aop->alipayrsaPublicKey = '<-- 请填写您的支付宝公钥，例如：MIIBIjANBg... -->';

// 使用支付宝公钥验签
$signVerified = $aop->rsaCheckV1($_POST, null, "RSA2");

if ($signVerified) {
    // 验签通过，解析业务参数
    $bizContent = json_decode($_POST['biz_content'], true);

    // 处理业务逻辑...

    // 必须返回纯文本 success 给支付宝，否则支付宝会重发通知
    echo "success";
}
else{
    // 验签失败
    echo "fail";
}

```
